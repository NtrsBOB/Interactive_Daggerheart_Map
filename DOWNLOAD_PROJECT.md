# How to download this project from GitHub

You’ve been given access to the map project. Use one of these ways to get it on your computer.

---

## Option 1: Download as a ZIP (easiest, no Git needed)

1. **Sign in to GitHub** in your browser: [github.com](https://github.com) (use the account you used to accept the invite).

2. **Open the project**  
   Use the link you were sent to the repo, or go to GitHub and find **Interactive_Daggerheart_Map** in your list of repositories.

3. **Download the code**  
   - Click the green **Code** button near the top of the page.  
   - Click **Download ZIP**.  
   - A file like `Interactive_Daggerheart_Map-main.zip` will download.

4. **Unzip the file**  
   - Find the ZIP in your Downloads folder (or wherever your browser saves files).  
   - Double-click it to unzip, or right-click → **Extract all**.  
   - You’ll get a folder (e.g. `Interactive_Daggerheart_Map-main`) with all the project files inside.

5. **Open the map**  
   - Go into that folder.  
   - Double-click **Interactive_Map.html** to open the map in your browser.  
   - **Map.png** must stay in the same folder so the map image loads.

**To get updates later:** download the ZIP again from GitHub and replace your folder (or merge the new files into your folder). You won’t get updates automatically with this method.

---

## Option 2: Clone with Git (if you use the command line)

If you have **Git** installed and are comfortable with a terminal:

1. Open a terminal (or Command Prompt / PowerShell) and go to the folder where you want the project (e.g. `cd Documents`).

2. Run (use the repo URL from the GitHub page — **Code** → copy the HTTPS link):

   ```bash
   git clone https://github.com/OWNER/Interactive_Daggerheart_Map.git
   ```

   Replace `OWNER` with the GitHub username of the person who owns the repo. They can send you the full URL.

3. Go into the new folder and open **Interactive_Map.html** in your browser.

**To get updates later:** open the project folder in the terminal and run `git pull`.

---

## Option 3: GitHub Desktop (app that handles Git for you)

1. **Install GitHub Desktop**: [desktop.github.com](https://desktop.github.com). Sign in with your GitHub account.

2. **Clone the repo**  
   - **File** → **Clone repository**.  
   - Choose the **URL** tab and paste the repo URL (from GitHub: **Code** → copy the HTTPS link), or pick the repo from the list if it appears.  
   - Choose a **local path** (e.g. Documents) and click **Clone**.

3. The project will be in the folder you chose. Open that folder and double-click **Interactive_Map.html** to use the map.

**To get updates later:** open the repo in GitHub Desktop and click **Fetch origin** / **Pull origin**.

---

## After you have the files

- **To view the map:** open **Interactive_Map.html** in your browser (double-click or drag into the browser). Keep **Map.png** in the same folder.
- **To edit locations:** see **INSTRUCTIONS.md** in the project folder.
